slurm_new.sh

This is the main file that executes the code parallely on different nodes of the cluster. 
Run this as:

sbatch slurm_new.sh
squeue -u reora

The results will be saved in 'results' folder.
The logs will be saved in the 'slurm_logs.sh' folder.
